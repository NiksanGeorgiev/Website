Front End Developer

HTML

HTML is the base level of the web development. Without understanding of HTML we can't take you because it will take too much time to get you through the basics which is not what we strive for.

CSS

No matter if you want to be front or back end developer you certainly need to know your way around CSS. You should be able to style any kind of element in the DOM and know how to use mediq queries.

JS

JavaScript is used in every website nowadays. It is essential that you know at least how to declare and deal with variables, flow control and loops. No other knowledge is required, but it would be helpful.

jQuery

We don't require you to know absolutely everything about jQuery but it would definately be in your favour if you have some prior knowledge in it. Don't worry it will come naturally one you understand JS.

Bootstrap

Some may say that Bootstrap is a bit outdated, but you will be surprised how much it is still used. Bootstrap will make your life as front developer 100 times easier and speed up your work by a lot.

Photoshop

As a front end developer it is rudementary to know how to use some kind of image manipulatior. It is not said that it should be Photoshop. It is all up to you and your preferences and likings.
